돔조작에 유용한 함수 및 이벤트

  • forEach() : 선택자(태그)의 갯수만큼 반복할 때
  • classList.add() : 클래스 추가
  • classList.remove() : 클래스 삭제
  • classList.toggle() : 클래스 토글
  • classList.contains() : 클래스 유무를 검사(true/false)
  • remove() : 요소를 삭제
  • insertAdjacentHTML() : 요소를 삽입(추가)
  • window.scrollTo() : 스크롤 이동
  • window.scrollY / window.scrollX : 스크롤값 얻기
  • window.getComputedStyle(선택자).css속성 : 선택자의 렌더링된 스타일값을 가져올 때
