DOM(Document Object Model)

  • HTML 문서의 구조와 그 정보를 말함
  • document.선택자.명령어/속성
  • getElementById("txt") : 아이디 선택자
  • getElementsClassName("red")[0] : 클래스 선택자
  • getElementsByTagName("li")[0] : 태그 선택자
  • querySelector("CSS 선택자")

주요속성

  • style: css속성을 가져오거나 변경
  • 요소의 속성명: 태그의 속성을 가져오거나 변경
  • innerText: 글자를 가져오거나 변경
  • innerHTML : 글자와 태그를 가져오거나 변경
