이벤트(Event)

  • 이벤트란 방문자가 입장이후의 모든 움직임(상호작용, 인터렉션)을 말함
  • onclick : 클릭
  • ondbclick : 더블클릭
  • onmouseover : 마우스가 올려졌을 때
  • ommouseout : 마우스가 벗어났을 때
  • onmousemove : 마우스가 움직일 때
  • onfocus : 포커스를 얻었을 때
  • onblur : 포커스를 잃었을 때
  • onkeypress : 키를 눌렀을 때, 특수키 안됨, 대소문자를 구분
  • onkeydown : 키를 눌렀을 때, 특수키 가능, 대문자로만 인식, 대부분 keydonw사용
  • onkeyup : 키를 떼었을 때
  • onchange : value값이 변경되었을 때
  • onload : 하위요소가 모두 로딩되었을 때
  • onresize : window/document의 사이즈가 변경되었을 때
  • onsubmit : 폼태그안에 전송기능이 작동될 때
  • onreset : 폼태그안에 있는 취소기능이 작동될 때
  • this 키워드 : 이벤트 발생한 요소를 가르키는 키워드
  • return 키워드 : 값을 반환
