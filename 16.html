jQuery 기본

  • 제이쿼리는 자바스크립트로 만든 라이브러리(프레임워크)
  • 라이브러리 : 함수(특정 기능)를 많들어 놓은 것, jQuery, css라이브러리
  • 프레임워크 : 작업환경을 제공하는 것
  • 공식사이트 : https://jquery.com/
  • 버전 1.x(구브라우저 호환), 버전 2.x(구브라우저 호환제외, 가벼워짐), 3.x(구브라우저 호환x, 기능강화)
  • uncompressed(원본), minified(압축), slim(간소화버전), slim minified(간소화 압축)
  • 선택자가 jQuery면 명령어도 jQuery여야함.

기본 메서드

  • css() : style속성을 변경/가져올 때
  • attr() : 요소의 속성을 변경/가져올 때
  • text() : 글자를 변경/가져올 때
  • html() : 글자+html를 변경/가져올 때
  • addClass / removeClasss() / toggleClass() : 클래스 추가/삭제/토글
  • click() : 클릭이벤트
  • eq(): 선택자의 인덱스번호로 재선택

수치관련

  • width() / height() : 폭/높이값을 적용/가져올 때
  • offset().top / left : 절대좌표값(top, left)을 반환
  • scrollTop() : 스크롤값을 얻거나 적용

객체(DOM) 편집 메서드

  • before() : 선택자 앞에 추가
  • after() : 선택자 뒤에 추가
  • prepend() : 선택자 첫 자식으로 추가
  • append() : 선택자 마지막 자식으로 추가
  • clone() : 선택자를 복사
  • remove(): 선택자를 삭제
  • empty() : 선택자 안의 내용을 비움

기타 메서드

  • val() : value
  • parent() : 선택자의 부모를 선택자
  • parents(".on") : 선택자 조상중에 클래스 on을 선택
  • children("a") : 선택자 자식중의 a태그를 선택
  • find(".on") : 선택자 자손중의 on이라는 클래스를 찾아라.
  • not(".on") : 선택자중에서 클래스 on을 제외
  • next() / prev() : 선택자 다음/이전형제 요소를 선택
  • hasClass("on") : 선택자에 클래스가 있으면 true
